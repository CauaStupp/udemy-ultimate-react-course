# Projetos — Ultimate React Course (Udemy)

Repositório que organiza os projetos que desenvolvo durante o curso **Ultimate React Course** (Udemy).

Cada pasta abaixo é um projeto ou exercício do curso. Clique para abrir a pasta específica e siga o `README.md` daquele projeto para instruções de execução e detalhes.

- [react-quiz-udemy](react-quiz-udemy/) — Aplicação de quiz em React + TypeScript (Vite). Veja o README dentro da pasta para como executar.

Como usar (exemplo):

```bash
# clonar o repositório
git clone <seu-repo-url>

# entrar na pasta do projeto desejado
cd react-quiz-udemy

# instalar dependências e rodar (recomendado pnpm)
pnpm install
pnpm dev
```

Se um projeto não tiver `README.md`, posso gerar automaticamente um com descrição, comandos e notas. Quer que eu gere READMEs individuais para todas as pastas de projeto?
